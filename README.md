# flaskr-devsecops-lab

Проект для лабораторной работы по DevSecOps.

## Уязвимости (для демонстрации)
- Hardcoded secret key
- Debug mode enabled
- SSTI (Server-Side Template Injection)
- Code injection (eval)
- Отсутствие аутентификации

## Инструменты
- Bandit (SAST)
- Trivy (SCA)
- Gitleaks (secrets detection)
